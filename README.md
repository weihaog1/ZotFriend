# ZotFriend

A swipe-based study buddy matching app for UC Irvine students.

## Features

- Swipe right to like, left to pass
- AI-powered match explanations (Gemini)
- Keyboard navigation (arrow keys)
- Match percentage scoring

## Run Locally

```bash
npm install
npm run dev
```

## Tech Stack

- React + TypeScript
- Framer Motion
- React Router
- Tailwind CSS
- Google Gemini API
